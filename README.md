# Artistic Portfolio (React)

This is a single-page portfolio with:

- artistic hero section
- about section
- projects section
- contact section

## Personalize quickly

Edit only one file:

- `src/content/siteContent.js`
- `public/assets/profile-art.png` (your portrait art file)

You can update your:

- name and hero text
- about paragraphs
- skills/bullets
- projects list
- contact links and email

## Run locally

```bash
npm install
npm start
```

## Deploy to GitHub Pages

```bash
npm run deploy
```
